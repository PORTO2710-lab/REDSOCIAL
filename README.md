# 📱 Red Social - Instagram Embed Service

Una implementación limpia y elegante para integrar publicaciones de Instagram en aplicaciones web.

## 🔗 Demo en vivo
Visualiza el servicio aquí: [https://porto2710-lab.github.io/REDSOCIAL/](https://porto2710-lab.github.io/REDSOCIAL/)

## 🚀 Características
* **Dark Mode Design:** Interfaz optimizada para reducir la fatiga visual.
* **Integración Nativa:** Mantiene todas las funcionalidades de la publicación original de Instagram.
* **Carga Optimizada:** Implementación de scripts asíncronos para mejorar el tiempo de respuesta.
* **Totalmente Responsivo:** Se adapta a dispositivos móviles y de escritorio.

## 🛠️ Stack Tecnológico
* **HTML5:** Estructura semántica del sitio.
* **CSS3:** Estilos personalizados y diseño de tarjetas.
* **JavaScript:** Manejo del renderizado dinámico de la red social.
